# 前言
  在當今的網路開發中，REST API 已成為許多應用程式和系統之間交互的主要方式。Actix Web，作為一個高效能的 Rust 網頁框架，提供了建立這些 API 的強大工具。

本文章將會帶你走過使用 Actix Web 建立 REST API 的整個過程，從專案的初始化到實作 Todo List 的 CRUD API。
